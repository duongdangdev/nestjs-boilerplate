# Project Nest.js Zero

## Environment

Node.js v20.10.0
MongoDb >= v5.0

## Installation

```bash
$ npm install
```

## Running the app

Duplicate file .env.example, rename it to .env and update value of environments variable

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
